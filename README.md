# Azure Security Monitoring Lab

## Overview

This project demonstrates how cloud environments can be monitored
for security threats using centralized logging and alerting.

The lab simulates security monitoring workflows used by
cloud security engineers and SOC analysts.

## Security Monitoring Goals

- Collect authentication and system logs
- Detect suspicious login activity
- Investigate alerts
- Document incident response steps

## Technologies Used

- Microsoft Azure
- Azure Monitor
- Microsoft Sentinel
- Python
- PowerShell
